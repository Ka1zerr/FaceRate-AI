# FaceRate AI (Umax Clone) 🌟

**FaceRate AI** — веб-приложение для анализа пропорций лица по фотографии, вдохновленное популярными приложениями для оценки внешности (Umax). Программа использует искусственный интеллект для нахождения ключевых точек на лице и математически вычисляет оценку на основе современных модельных стандартов красоты (looksmaxxing).

> **v2.0** — Добавлена 3D-нормализация ландмарков и компенсация позы головы.
> Теперь оценка стабильна на селфи, при повороте головы и разных условиях съёмки.

---

## 📸 Скриншоты и Интерфейс
Интерфейс выполнен в **темном минималистичном стиле**. Пользователь может загрузить селфи с помощью Drag & Drop или выбрав файл, и получить анализ без перезагрузки страницы. Никакого визуального шума — только чистая и стильная функциональность.

---

## 🧠 Как работает логика оценивания?

Программа **не использует** субъективные оценки "красиво/некрасиво". В основе лежит строгая 3D-геометрия и статистика:

1. **Детекция лица:** Фотография отправляется на backend, где нейросеть `MediaPipe FaceLandmarker` от Google наносит на лицо **478 ключевых 3D-точек (landmarks)**.
2. **3D-нормализация и компенсация позы:**
   * Из 3D-координат ландмарков (x, y, z) оценивается **поза головы** (yaw, pitch, roll) через алгоритм `solvePnP`.
   * Ландмарки математически **проецируются во фронтальную плоскость** — лицо "разворачивается" в анфас, убирая перспективное искажение от селфи-камеры и поворот головы.
   * Если голова повёрнута слишком сильно (>15°), приложение выдаёт **предупреждение о качестве**.
3. **Извлечение метрик:** Из нормализованных 3D-координат вычисляются 6 ключевых параметров:
   * **FWHR (Ширина лица):** Отношение ширины лица к высоте средней трети. У моделей этот показатель высокий (>1.85).
   * **Canthal Tilt (Наклон глаз):** Угол между внутренним и внешним уголками. "Хищный" взгляд (3°-8°) ценится выше.
   * **Hunter Eyes (Разрез глаз):** Отношение высоты глаза к ширине. Чем уже прищур, тем выше балл.
   * **Jaw-to-Cheek Ratio (Челюсть):** Отношение ширины челюсти к ширине скул.
   * **Lower Third (Нижняя треть):** Подбородок должен занимать ~33-34% от высоты лица.
   * **Symmetry (Симметрия):** Сравнение левой и правой половин лица.
4. **Статистический анализ:**
   * Каждое значение оценивается относительно **среднего в популяции** и **идеала**.
   * Среднее лицо получит `5.0`. Для `7.0+` нужно быть лучше 80% людей. Для `9.0+` — топ 1%.

## 🆕 Что нового в v2.0

| Было (v1) | Стало (v2) |
|---|---|
| Сырые 2D-координаты из кадра | 3D-координаты с компенсацией позы |

## 🛠 Технологический стек

* **Бэкенд:** Python + [FastAPI](https://fastapi.tiangolo.com/)
* **Компьютерное зрение (AI):** [MediaPipe Tasks API](https://developers.google.com/mediapipe/solutions/vision/face_landmarker) (478 3D-ландмарков)
* **3D Pose Estimation:** OpenCV `solvePnP` для оценки yaw/pitch/roll
* **Обработка изображений:** OpenCV, Pillow
* **Сервер:** Uvicorn (ASGI)
* **Фронтенд:** HTML5 + CSS3 + JavaScript (Fetch API). Без тяжёлых фреймворков.

## 🚀 Установка и запуск

Предполагается Python 3.9+.

1. **Клонируйте репозиторий:**
   ```bash
   git clone https://github.com/Ka1zerr/FaceRate-AI.git
   cd FaceRate-AI
   ```
2. **Установите зависимости:**
   ```bash
   pip install -r requirements.txt
   ```
3. **Скачайте модель (если нет в папке):**
   Файл `face_landmarker.task` должен лежать рядом с `main.py`. Скачать: [face_landmarker.task](https://storage.googleapis.com/mediapipe-models/face_landmarker/face_landmarker/float16/1/face_landmarker.task)
4. **Запустите сервер:**
   ```bash
   python -m uvicorn main:app --host 0.0.0.0 --port 8000
   ```
5. **Откройте:** `http://localhost:8000`
