# FaceRate AI (Umax Clone) 🌟

**FaceRate AI** — это веб-приложение для анализа пропорций лица по фотографии, вдохновленное популярными приложениями для оценки внешности (такими как Umax). Программа использует искусственный интеллект для нахождения ключевых точек на лице и математически вычисляет оценку на основе современных модельных стандартов красоты (looksmaxxing).

---

## 📸 Скриншоты и Интерфейс
Интерфейс выполнен в **премиальном темном минималистичном стиле** (с контрастными цинковыми/белыми элементами и крутой конструктивистской типографикой Rodchenko + Inter). Пользователь может загрузить селфи с помощью Drag & Drop или выбрав файл, и получить анализ без перезагрузки страницы. Никакого визуального шума — только чистая и стильная функциональность.

---

## 🧠 Как работает логика оценивания?

Программа **не использует** субъективные оценки "красиво/некрасиво". В основе лежит строгая геометрия и статистика:

1. **Детекция лица:** Фотография отправляется на backend, где нейросеть `MediaPipe FaceLandmarker` от Google наносит на лицо **478 ключевых точек (landmarks)**.
2. **Извлечение метрик:** Из координат этих точек вычисляются физические расстояния и углы. Мы анализируем 6 ключевых параметров:
   * **FWHR (Ширина лица):** Отношение ширины лица к высоте средней трети. У моделей (chad face) этот показатель высокий (>1.85).
   * **Canthal Tilt (Наклон глаз):** Угол между внутренним и внешним уголками глаз. "Хищный" взгляд (позитивный наклон 3°-8°) ценится выше.
   * **Hunter Eyes (Разрез глаз):** Отношение высоты глаза к его ширине. Чем уже прищур (меньше значение), тем выше балл.
   * **Jaw-to-Cheek Ratio (Челюсть):** Отношение ширины нижней челюсти к ширине скул. Широкая квадратная челюсть дает максимум очков.
   * **Lower Third (Нижняя треть):** Идеальный подбородок должен занимать ровно ~33-34% от высоты всего лица.
   * **Сymmetry (Симметрия):** Сравнение левой и правой половин лица относительно центра.
3. **Строгий статистический анализ (Z-Score):**
   * Полученные значения не оцениваются линейно. Они прогоняются через формулу вычисления вероятности на основе **нормального распределения (Колоколообразная кривая)**.
   * Программа "знает" средние значения этих черт в популяции людей.
   * Идеально среднее лицо получит ровно `5.0`.
   * Чтобы получить `7.0+`, черта должна быть лучше, чем у 80% людей.
   * Чтобы получить `9.0 - 10.0`, ваши пропорции должны входить в топ 1% (модельная внешность).

## 🛠 Технологический стек

* **Бэкенд:** Python + [FastAPI](https://fastapi.tiangolo.com/) (очень быстрый и современный микрофреймворк).
* **Компьютерное зрение (AI):** [MediaPipe Tasks API](https://developers.google.com/mediapipe/solutions/vision/face_landmarker) (позволяет с высокой точностью находить глаза, нос, губы и контуры челюсти).
* **Обработка изображений:** OpenCV (`opencv-python-headless`), Pillow.
* **Сервер:** Uvicorn (ASGI-сервер для Python).
* **Фронтенд:** Чистый HTML5, CSS3 (Строгий минимализм, CSS-переменные) и JavaScript (с использованием Fetch API для асинхронных запросов). Никаких тяжелых JS-фреймворков.

## 🚀 Установка и запуск (Локально)

Предполагается, что у вас установлен Python версии 3.9+.

1. **Клонируйте репозиторий или скачайте папку с проектом.**
2. **Перейдите в папку проекта в терминале:**
   ```bash
   cd path/to/face_app
   ```
3. **Установите зависимости:**
   ```bash
   pip install -r requirements.txt
   ```
4. **Скачайте модель MediaPipe (опционально):**
   Модель `face_landmarker.task` уже должна быть в папке проекта. Если ее нет, приложение попытается скачать ее самостоятельно, или вы можете скачать её [отсюда](https://storage.googleapis.com/mediapipe-models/face_landmarker/face_landmarker/float16/1/face_landmarker.task) и положить рядом с `main.py`.
5. **Запустите сервер:**
   ```bash
   python -m uvicorn main:app --host 0.0.0.0 --port 8000
   ```
6. **Откройте приложение:**
   Перейдите в браузере по адресу `http://localhost:8000` или `http://127.0.0.1:8000`.
